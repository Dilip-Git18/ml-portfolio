## 📌 About This Project

This repository contains my **personal portfolio website**, built to showcase my journey as a
**B.Tech Computer Science student** and my growing skills in **Machine Learning** and
**Full-Stack Web Development**. It was created as part of the **FlyRank AI Internship**, where I
worked on applied ML projects and strengthened my end-to-end development skills.

The site is a single-page, fully responsive portfolio built with **pure HTML, CSS, and
JavaScript** — no frameworks — and is deployed live on **Vercel**.

## ✨ Features

- 🏠 **Home** — Hero introduction with role, headline, and quick call-to-action buttons
- 👤 **About Me** — Short bio, education, and background
- 🛠️ **Skills** — Filterable, categorized skill cards (Languages, ML, Tools)
- 💼 **Projects** — Expandable project cards with Problem → Solution → Outcome, screenshots,
  GitHub links, and live demo links
- 📜 **Certifications** — Anthropic Academy, FlyRank AI Internship, AWS Certifications
- 📩 **Contact** — Direct email, GitHub, LinkedIn, and a working contact form (via FormSubmit)
- 🌙 **Dark / Light mode** toggle with saved preference
- 📱 **Fully responsive** across mobile, tablet, and desktop
- 🎞️ **Smooth scroll animations** and an animated neural-network hero background
- 📄 **Resume preview** — opens directly in-browser instead of forcing a download

## ⚙️ Installation & Local Setup

No build tools, package managers, or dependencies are required — this project runs entirely in
the browser.

1. **Clone the repository**
   ```bash
   git clone https://github.com/Dilip-Git18/ml-portfolio.git
   ```

2. **Move into the project folder**
   ```bash
   cd ml-portfolio
   ```

3. **Open it in your browser**
   - Simply double-click `index.html`, **or**
   - Use a lightweight local server (recommended, especially for the contact form):
     ```bash
     npx serve .
     ```
     or, with the VS Code "Live Server" extension, right-click `index.html` → **Open with Live Server**.

4. **Deploy your own copy (optional)**
   - Push the repository to your GitHub account.
   - Import it into [Vercel](https://vercel.com/) as a static site — no build configuration needed.
